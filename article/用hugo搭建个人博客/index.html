Windows中用hugo快速搭建个人博客

真正的极客必然拥有自己的博客，hugo是一个用go语言编写的程序，搭建起来极为简单。

工作准备

  • 下载hugo。
    在GitHub中，下载对应windows版本的hugo，网址：https://github.com/gohugoio/hugo/releases

  • 解压文件
    将下载的文件解压后，将hugo.exe所在的目录配置到系统变量path中。

  • 验证
    调出命令面板，运行hugo help，如果有正确的帮助信息出现，则说明安装完成。

本地部署hugo

hugo new site myblog(新建文件夹名) 部署成功后应该出现myblog文件夹，其中应该有以下几个文件夹

     <DIR>          archetypes
                 82 config.toml
     <DIR>          content
     <DIR>          data
     <DIR>          layouts
     <DIR>          resources
     <DIR>          static
     <DIR>          themes

选择主题

主题商店：https://themes.gohugo.io/ 在主题页面找到安装主题的git命令，且git克隆主题

启动服务

hugo server -t +主题名 –buildDrafts（此命令要在刚才新建的myblog根目录执行）

创建博客

hugo new post/文件名.md（此命令要在刚才新建的myblog根目录执行）

将博客部署到GitHub(通过GitHub直接访问博客)

  1. 提前准备 在GitHub上建立好仓库，仓库名是以后要访问的链接，如：zhangxiaoshuai98.github.io/
  2. 生成要推送到GitHub的html等文件 hugo --theme=(主题名) --baseUrl="https://zhangxiaoshuai98.github.io/" --buildDrafts 此命令执行后，会出现一个public文件夹，我们只需要将这个文件夹用git推送到GitHub上即可
  3. 将public文件夹内容上传到GitHub上
cd public
git init
git add .
git commit -m "hugo博客第一次提交"
git remote add origin https://github.com/zhangxiaoshuai98/zhangxiaoshuai98.github.io.git
git push -u origin master

这样我们就成功部署到了GitHub上，之后可以直接通过浏览器访问https://zhangxiaoshuai98.github.io/访问
